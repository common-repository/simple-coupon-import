== Description ==
<strong>Simple Coupon Import</strong>
<br />
- Plugin hổ trợ theme <a href="https://www.famethemes.com/themes/coupon-wp/" target="_blank">Coupon WP</a> <br/>
- Nhập liệu nhanh bằng file CSV, và đặc biệt dành cho các publisher làm affiliate import dữ liệu từ nguồn <a href="https://www.famethemes.com/themes/coupon-wp/" target="_blank">AccessTrade</a>


<h4>♥️ Tính năng plugin </h4>
<ul>
    <li><strong>Upload file CSV</strong> nhập liệu bằng file</li>
    <li><strong>Api AccessTrade</strong> kết nối api get danh sách coupon</li>
</ul>

<a href="https://sanphamtienich.com/" target="_blank"><strong>View Demo</strong></a>

== Installation ==
1. Upload the entire plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
